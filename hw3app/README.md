
  # Figma Application

  Figma AI project: https://www.figma.com/design/iqihNK7aDouzGeWeG1NaWN/Multi-Page-React-Application.

  ## Running the front end

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Running the backend
  
  Navigate to the backend folder. 

  First start your own python env. Ex: `source path_to_pythonEnv/bin/activate` 

  Then install requirements: `python3 -m pip install -r requirements.txt`

  Finally, run `python3 -m uvicorn main:app --reload` to start the python backend.
